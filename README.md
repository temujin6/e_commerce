Доорх DB_PASSWORD, DB_NAME гэсэн хувьсагч дээр өөрийн Database-ийн нэр болон нууц үгийг оруулна.

.env файл

DB_HOST=localhost

DB_USER=root

DB_PASSWORD=

DB_NAME=e_commerce

npm run dev
